# 个人知识管理系统 (Personal Knowledge Management System)

这是一个基于HTML、CSS和JavaScript的个人知识管理系统原型，具有现代化的用户界面和丰富的功能。

## 功能特性

1. **学习概览面板**
   - 展示关键学习指标（今日学习时长、已学文档数等）
   - 推荐学习路径
   - 最近上传的文档展示

2. **文档管理面板**
   - 文档分类浏览
   - 多种筛选选项（类别、状态等）
   - 视图切换（网格/列表）
   - 排序功能

3. **知识图谱面板**
   - 可交互的知识点关系图
   - 使用ECharts可视化库
   - 知识点详情展示

4. **学习分析面板**
   - 学习时长统计图表
   - 知识点掌握情况仪表盘
   - 学习进度预测和建议

5. **AI助手面板**
   - 与AI助手对话界面
   - 提问和回答功能

## 技术栈

- HTML5
- Tailwind CSS
- JavaScript (ES6+)
- ECharts 数据可视化库
- Iconify 图标库

## 如何运行

1. 克隆或下载项目文件
2. 在浏览器中打开 `index.html` 文件即可

或者使用本地服务器运行：

```bash
# 如果安装了Python
python -m http.server 8000

# 然后在浏览器中访问 http://localhost:8000
```

## 功能说明

### 导航系统
- 通过左侧导航栏在不同功能模块间切换
- 当前选中模块会高亮显示

### 响应式设计
- 支持桌面端和移动端显示
- 在小屏幕设备上会自动调整布局

### 交互功能
- 所有按钮和控件都有相应的交互反馈
- 图表支持缩放和拖拽操作
- 表单元素具有基本的验证功能

## 开发说明

该项目是一个前端静态原型，展示了完整的用户界面和交互逻辑。在实际生产环境中，需要连接后端服务来提供数据存储和业务逻辑处理。

主要的JavaScript功能包括：
- 视图切换管理
- 图表初始化和更新
- 表单控件绑定事件
- 数据模拟和展示
